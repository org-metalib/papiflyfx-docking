# Ribbon 10 Side Toolbar Placement

Ribbon 10 replaces the Ribbon 9 persistent side command pane for `LEFT` and
`RIGHT` placement with a compact outside side-toolbar rail plus transient
command popovers over dock content. The public placement API, provider SPI, and
`extensions.ribbon.placement` session field remain unchanged.

- [concept.md](concept.md) - target UX, compatibility constraints, and product decisions.
- [plan.md](plan.md) - implementation phases, invariants, reviewers, and validation strategy.
- [progress.md](progress.md) - implementation status, validation evidence, and handoff notes.
