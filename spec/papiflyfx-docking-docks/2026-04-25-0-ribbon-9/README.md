# Docks Ribbon Side Placement

**Lead Agent:** @core-architect
**Design Support:** @ui-ux-designer
**Validation:** @qa-engineer
**Spec Steward:** @spec-steward

## Summary

Ribbon 9 extends ribbon hosting beyond the current top-only placement. Hosts should be able to place the ribbon 
on the top, left, right, or bottom side of the dock host while preserving command discovery, contextual tabs,
Quick Access Toolbar state, minimized behavior, adaptive layout, and session persistence.
