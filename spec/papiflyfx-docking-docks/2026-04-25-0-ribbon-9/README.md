# Docks Ribbon Side Placement

**Lead Agent:** @core-architect
**Design Support:** @ui-ux-designer
**Validation:** @qa-engineer
**Spec Steward:** @spec-steward

## Summary

Ribbon 9 extends ribbon hosting beyond the current top-only placement. Applications should be able to host the ribbon
on the top, bottom, left, or right side of the dock host when that placement is specified by the host configuration,
while preserving command discovery, contextual tabs, Quick Access Toolbar state, minimized behavior, adaptive layout,
and session persistence.
