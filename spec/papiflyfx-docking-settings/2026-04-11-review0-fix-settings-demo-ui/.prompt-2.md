# Agent 2 — `@feature-dev`: Make SamplesApp Theme-Aware

**Module scope:** `papiflyfx-docking-samples` only (read `papiflyfx-docking-api` for reference, do not modify it)

## Context

`SamplesApp.java` uses hardcoded dark-only hex colors for the top bar (`#3c3c3c`), content area (`#1e1e1e`),
navigation category headings (`#1e1e1e`, `#888888`), buttons (`#5b6470`, `#0e639c`), and labels (`#bbbbbb`).
The `NAVIGATION_THEME` is a static final dark-only `TreeViewTheme`. None of these respond to the
light/dark theme toggle, creating a broken experience when switching themes.

The fix uses `UiCommonThemeSupport` helpers from `papiflyfx-docking-api` to derive colors from the active `Theme`.

## Task 1 — Make top bar theme-aware

**File:** `papiflyfx-docking-samples/src/main/java/org/metalib/papifly/fx/samples/SamplesApp.java`

### 1a. Promote `titleLabel`, `authHint`, and `topBar` HBox to fields

They need to be re-styled when the theme changes.

### 1b. Add an `applyTopBarTheme` method

```java
private void applyTopBarTheme(Theme theme) {
    Color bg = UiCommonThemeSupport.headerBackground(theme);
    Color text = UiCommonThemeSupport.textPrimary(theme);
    Color muted = UiCommonThemeSupport.alpha(text, 0.66);
    Color accent = UiCommonThemeSupport.accent(theme);
    topBar.setStyle("-fx-background-color: " + UiCommonThemeSupport.paintToCss(bg, "#3c3c3c") + ";");
    titleLabel.setStyle("-fx-text-fill: " + UiCommonThemeSupport.paintToCss(text, "white")
        + "; -fx-font-size: 14px; -fx-font-weight: bold;");
    authHint.setStyle("-fx-text-fill: " + UiCommonThemeSupport.paintToCss(muted, "#bbbbbb")
        + "; -fx-font-size: 12px;");
}
```

### 1c. Make `styleTopBarButton` theme-aware

Replace the hardcoded background colors with theme-derived accent/secondary colors:

```java
private void styleTopBarButton(Labeled button, boolean primary) {
    Theme theme = themeProperty.get();
    Color accent = UiCommonThemeSupport.accent(theme);
    Color bg = primary ? accent : UiCommonThemeSupport.alpha(accent, 0.40);
    button.setStyle("-fx-background-color: " + UiCommonThemeSupport.paintToCss(bg, "#0e639c")
        + "; -fx-text-fill: white;");
}
```

Update callers:
- `authSettingsButton` → `styleTopBarButton(authSettingsButton, false)`
- `loginDemoButton` → `styleTopBarButton(loginDemoButton, true)`
- `themeToggle` → `styleTopBarButton(themeToggle, true)`

### 1d. Add theme listener for top bar

In `start()`, after building the top bar:

```java
themeProperty.addListener((obs, oldTheme, newTheme) -> {
    applyTopBarTheme(newTheme);
    applyContentAreaTheme(newTheme);
    updateNavigationTheme(newTheme);
    // re-style buttons
    styleTopBarButton(authSettingsButton, false);
    styleTopBarButton(loginDemoButton, true);
    styleTopBarButton(themeToggle, true);
});
```

### 2b. Update `buildContentArea`

Replace the hardcoded placeholder label style with theme-derived muted text:

```java
private void buildContentArea() {
    Label placeholder = new Label("Select a sample from the navigation panel...");
    Theme theme = themeProperty.get();
    Color muted = UiCommonThemeSupport.alpha(UiCommonThemeSupport.textPrimary(theme), 0.66);
    placeholder.setStyle("-fx-text-fill: " + UiCommonThemeSupport.paintToCss(muted, "#888888")
        + "; -fx-font-size: 14px;");
    placeholder.setWrapText(true);
    placeholder.setMaxWidth(420);
    contentArea.getChildren().add(placeholder);
    applyContentAreaTheme(theme);
}
```

## Task 3 — Make navigation tree theme-aware

**File:** same `SamplesApp.java`

### 3a. Replace static `NAVIGATION_THEME` with a method

```java
private static TreeViewTheme buildNavigationTheme(Theme theme) {
    Color bg = UiCommonThemeSupport.background(theme);
    Color text = UiCommonThemeSupport.textPrimary(theme);
    Color accent = UiCommonThemeSupport.accent(theme);
    Color border = UiCommonThemeSupport.border(theme);
    Color hover = UiCommonThemeSupport.hover(theme);
    // Build a TreeViewTheme using theme-derived colors instead of hardcoded values
    return new TreeViewTheme(
        bg, bg, bg,                                         // background, backgroundOdd, backgroundEven
        UiCommonThemeSupport.alpha(accent, 0.30),           // selectionBackground
        hover,                                              // hoverBackground
        accent,                                             // focusBorder
        UiCommonThemeSupport.alpha(text, 0.08),             // indentGuide
        text,                                               // textColor
        UiCommonThemeSupport.textActive(theme),             // textColorSelected
        UiCommonThemeSupport.alpha(text, 0.50),             // textColorDisabled
        border,                                             // separator
        UiCommonThemeSupport.alpha(text, 0.08),             // scrollThumb
        UiCommonThemeSupport.alpha(text, 0.32),             // scrollThumbHover
        UiCommonThemeSupport.alpha(text, 0.46),             // scrollThumbPressed
        UiCommonThemeSupport.alpha(text, 0.58),             // scrollThumbDragging
        Font.font("System", 13),
        24.0, 16.0, 0.0
    );
}
```

### 3b. Add `updateNavigationTheme` method

```java
private void updateNavigationTheme(Theme theme) {
    sampleTree.setTreeViewTheme(buildNavigationTheme(theme));
}
```

### 3c. Update `renderNavigationCell`

Replace hardcoded category colors:

if (entry.isCategory()) {
